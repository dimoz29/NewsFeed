# Greek Newsfeed

Free Greek news aggregator (50+ sources) built with React + Tailwind, hosted on Cloudflare Pages.

## Development
- Install deps: `npm i`
- Run locally: `npm run dev`
- Build: `npm run build`

## Deploy
- Connect this repo to Cloudflare Pages (Framework preset: Vite)
- Build command: `npm run build`
- Output dir: `dist`
